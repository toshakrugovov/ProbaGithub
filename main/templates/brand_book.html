🎨 Брендбук MPTCOURSE

2. Типографика

В качестве основного шрифта используется Inter — современный sans-serif шрифт, разработанный специально для интерфейсов. Inter обеспечивает отличную читаемость на всех размерах экранов, поддерживает кириллицу и имеет оптимизированные метрики для веб-приложений. Иерархия типографики построена на чётком разделении уровней заголовков и текста.

3. Сетка и отступы

В основе компоновки интерфейса лежит 12-колоночная сетка с максимальной шириной контейнера 1200px. Это обеспечивает гибкость при создании различных раскладок и единообразие на всех страницах. Система отступов построена на базе 8px (0.5rem) для создания ритмичной и гармоничной композиции.

6. Адаптивность

Интерфейс MPTCOURSE полностью адаптивен и оптимизирован для всех типов устройств. Система использует Mobile First подход с тремя основными брейкпоинтами: мобильные устройства (до 768px), планшеты (768-1024px) и десктопы (от 1024px). На каждом брейкпоинте обеспечивается оптимальная раскладка и удобство использования.

8. Анимации и переходы

Все интерактивные элементы интерфейса MPTCOURSE имеют плавные переходы и микро-анимации. Это улучшает восприятие действий пользователя, делает интерфейс более отзывчивым и современным. Стандартная длительность анимаций — 0.2-0.3 секунды, что обеспечивает быструю реакцию без ощущения задержки.

/* Стандартные переходы */
transition: all 0.2s ease;
transition: transform 0.15s ease, box-shadow 0.2s ease;

/* Hover эффекты */
transform: translateY(-1px);
box-shadow: 0 6px 16px rgba(0,0,0,0.2);

9. Применение стандартов

При разработке новых компонентов, страниц или функциональности интернет-магазина MPTCOURSE необходимо строго следовать следующим правилам:

  • Цвета: Использовать только цвета из утверждённой палитры через CSS-переменные (var(--bg), var(--text), var(--surface) и т.д.) для обеспечения поддержки светлой и тёмной тем
  • Типографика: Применять шрифт Inter и следовать иерархии размеров и весов (display, heading 1-3, body, small)
  • Сетка: Использовать 12-колоночную сетку с максимальной шириной контейнера 1200px и соблюдать систему отступов (кратность 8px)
  • Адаптивность: Обеспечивать корректное отображение на всех брейкпоинтах (mobile, tablet, desktop) с использованием медиа-запросов
  • Интерактивность: Добавлять плавные переходы (transition) для всех интерактивных элементов (кнопки, ссылки, карточки) с длительностью 0.2-0.3 секунды
  • Тестирование: Обязательно проверять отображение в светлой и тёмной темах, а также на разных размерах экранов
  • Доступность: Обеспечивать достаточный контраст текста, использовать семантические HTML-теги и добавлять альтернативные тексты для изображений
  • Производительность: Оптимизировать изображения, минимизировать количество HTTP-запросов и использовать CSS-переменные для эффективного кеширования стилей

Важно

Любые отклонения от стандартов должны быть обоснованы и согласованы. При возникновении вопросов обращайтесь к этому документу или к команде разработки.
